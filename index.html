Why Use DSL?

  • Its easier to express ideas in a higher level language

Types of DSL's

  • Embedded (internal)
    • Resides within syntax of host language
  • Standalone (external)
    • Specialized / Independent syntax

DSL Exmaples

  • HTML, CSS
  • SQL
  • ANT, Maven, Make
  • Shellbe

External Tools

  • Antlr
  • YACC (Yes Another Complier Compiler)

Uses a formal grammar (DSL) to build

Internal Tools

  • Parser Combinators
  • Recursive descent parser

DSL Flow

  • Represent ideas in one formate and translate to another
    • Compiling to another formate
    • Evaluating for results

Examples of compilations

  • Java -> Byte Code
  • Byte Code -> Machine Code

Examples of Evaluation

  • Machine Code -> Hello World
  • HTML -> Web Page
